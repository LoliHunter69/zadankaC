masa: liczba atomowa
ladunek: jednostki atomowe
wspolrzedne: angstromy

a nastepnie program, ktory na podstawie struktur wyznaczy srodek masy dwoch atomow oraz potencjal dla ukladu tych atomow oraz potencjal dla ukladu
